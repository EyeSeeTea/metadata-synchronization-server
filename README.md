## Setup

```
$ yarn install
```

File `app-config.json` must be created. If no configuration file is supplied the following is used as a placeholder:

```json
{
    "encryptionKey": "encryptionKey",
    "apiUrl": "https://play.dhis2.org/2.30/api",
    "username": "admin",
    "password": "district"
}

```

## Development

Start development server:

```
$ yarn start
```

## Build

```
$ yarn build
```
